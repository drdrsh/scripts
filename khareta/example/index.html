Khareta : A Google Maps jQuery Plugin

Khareta (خريطة) means "Map" in Arabic, It is a jQuery plugin that transforms your average textarea into a Google Maps Input component where the user can place points, polylines or polygons on the map and have the result serialized to the textarea immediately.
For documentation please visit Khareta on GitHub

Using the examples

You can pick a mode from the lower left corner of the map then click the map to add points, to remove points just click the markers again, to move existing points around drag the point's marker.

To allow yourself more freedom click the "Full Screen" button on the upper right corner of the screen to go in Fullscreen mode that will allow you more space to place your points. to go back to the normal mode click the "Full Screen" button again.

Simple Example

This is the simplest possible example for Khareta acheived by this code:

$('#textarea_1').khareta();

o see the internal data of this map, open your console and write

$('#textarea_1').khareta('getData');
